# Pulse

SaaS metrics dashboard. Stripe-native. Nothing extra.
Four numbers. One screen. €34.99/mo.

Built by one person, in public.

## What it does

- Where your paying customers actually came from (UTM → Stripe, genuinely connected)
- Trial to paid % this month
- Landing page views with first-party tracking (works even when uBlock kills GA)
- Weekly revenue cohorts

## Why I built it

I found a post from a founder at $11k MRR paying $200/mo across three 
tools that don't talk to each other, with someone spending three hours 
every Tuesday manually pulling Stripe, Mixpanel and Plausible into a 
Google Sheet. Just to see four numbers.

The tool that should have existed didn't. So I'm building it.

## Stack

- Next.js
- Stripe API
- Postgres
- Claude Code + Antigravity for development
